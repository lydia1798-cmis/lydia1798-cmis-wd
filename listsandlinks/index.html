How to create a new webpage on the production server!!!

  1. First you need to create a html file as well as a css file.
  2. You then need to code your html and css files to make your webpage (as you desire).
  3. Start your development server to view (and check) your work.
  4. Now you need to add, commit and push those files to you repository on github.com.
  5. Start SFTP by typing the command: sftp username1234@cmis-hsteacher, and then signing in.
  6. Go into the folder public.html.
  7. Put both your css and html files into the public html folder.
  8. View your website from the production server using the url: cmis-hsteacher/~username1234/your-file-name.
